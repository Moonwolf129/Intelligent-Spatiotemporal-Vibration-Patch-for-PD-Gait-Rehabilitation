# Rehealthy Android App

This is a minimal Android/Kotlin project for the intelligent spatiotemporal vibration patch.

Features (skeleton):

- Scan and connect to the patch via BLE (device name: `ReHealthyDevice`).
- Subscribe to IMU / gait metrics notifications (placeholders).
- Show basic real-time information on a dashboard screen.

Import `RehealthyApp` into Android Studio as an existing project.
